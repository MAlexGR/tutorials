Ορολογία
#########

.. glossary::
    **Tutorial**
        Εκπαιδευτικό υλικό ή μάθημα ή διδασκαλία (χρησιμοποιήστε όποιον όρο θέλετε). Έχει σκοπό να μεταδώσει γνώσεις για κάποιο αντικείμενο/πεδίο/τομέα, με περισσότερο διαδραστικό τρόπο από ότι ένα βιβλίο (book) ή μία διάλεξη (lecture). Συνήθως ένα tutorial περιλαμβάνει παραδείγματα ή/και πληροφορίες, για να υλοποιηθεί κάποια συγκεκριμένη εργασία/σκοπός.

        Ένα tutorial δεν επεκτείνεται σε πολλές λεπτομέρειες πάνω στο γνωστικό πεδίο του αντικειμένου του. Είναι περισσότερο μία σειρά βημάτων που καθοδηγούν κάποιον μη ειδικό ή αρχάριο στο να κατανοήσει κάτι. Από αυτό συνάγεται ότι ένα tutorial έχει σκοπό την *εκμάθηση* (learning-oriented).

        Μπορεί να έχει έντυπη η ψηφιακή μορφή (ηλεκτρονικά έγγραφα, video, ιστοσελίδες κ.λπ.).

        Λήφθηκαν πληροφορίες από `Wikipedia "Tutorial" <https://en.wikipedia.org/wiki/Tutorial>`_ και άλλες δικτυακές πηγές.

    **How To**
        Ανεπίσημος και συνήθως σύντομος οδηγός ή εκπαιδευτικό υλικό, που περιγράφει με σαφή τρόπο πως να αντιμετωπισθεί μία συγκεκριμένη κατάσταση/θέμα/αντικείμενο/εργασία.

        Λήφθηκαν πληροφορίες από `Wikipedia "How to" <https://en.wikipedia.org/wiki/How-to>`_ και άλλες δικτυακές πηγές.
