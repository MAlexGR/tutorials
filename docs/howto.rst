Q&A
========

Σε αυτό το τμήμα θα ασχοληθούμε με βασικές ερωτήσεις και απαντήσεις πάνω σε σημαντικά θέματα. Θα χρησιμοποιήσουμε για συντομία τη γνωστή αγγλική συντομογραφία "Q&A" κυρίως για λόγους τυποποίησης, όταν αναφερόμαστε στο συγκεκριμένο θέμα.

Σε αντίθεση όμως με τη κοινή πεποίθηση που υπάρχει για τις *ερωταπαντήσεις*, που αφορούν σε συγκεκριμένα προβλήματα και πως μπορούμε να τα αντιμετωπίσουμε, εδώ θα προσπαθήσουμε να απαντήσουμε (κυρίως) σε γενικότερα ερωτήματα όπως "γιατί χρειάζεται η τεκμηρίωση?" ή "που χρησιμεύει ο έλεγχος εκδόσεων?" κ.λπ. Ίσως τώρα δε γίνονται εύκολα κατανοητά τέτοια ερωτήματα αλλά η χρησιμότητά τους θα φανεί στη συνέχεια.
