Έναρξη
--------

Γιατί "Αθηνά";

Για έναν απλό, "ρομαντικό", ουσιαστικό και επιδεικτικό λόγο:

   **ΑΘΗΝΑ**
   
   Η θεά της Δύναμης και Σοφίας!

Τι σημαίνει αυτό; Πρώτα από όλα αποφάσησε αν "θέλεις". Αν "λυπάσαι", "δεν μπορείς", "...που να τρέχεις τώρα!" και λοιπά συναφή, μείνε σπίτι και κοιμίσου. Αν πραγματικά "θέλεις" και είσαι έτοιμος, να ξέρεις ότι σίγουρα "μπορείς"! Θα προσπαθήσω να σου δείξω πως. Αυτός ο ιστοχώρος απευθύνεται σε ανθρώπους που θέλουν να μάθουν (γνώση, σοφία) και είναι έτοιμοι να το κάνουν (δύναμη) για να δημιουργήσουν (μπορώ).

Σε αυτό το site παρουσιάζουμε μία σειρά με οδηγούς για αντικείμενα που έχουν σχέση με τον κόσμο της πληροφορικής.
